# CSE315_library_checkout
Library Checkout System written in both Rust and Python using polymorphic behavior.

To run Python: Inside python_checkout run '<Your python version> __main__.py'
To test Python: Inside python_checkout run 'pytest'

To run Rust: Inside rust_checkout run 'cargo run'
To test Rust: Inside rust_checkout run 'cargo test'

Requires Rust, Python, Cargo, and Pytest
